How it works

Zephyr turns your question into a validated PlanV2, compiles fetch steps to GraphQL, and executes them against the backend when enabled.

How To Ask

For the best results, be specific about time, entity, and scope.

1. Add a time window when it matters: on 2026-02-17, last week.
2. Put exact lookup values in quotes: "the wagon", "Turbine 2", "Wind Farm 1".
3. Ask for one clear task at a time: trend, compare, nearest, list, or filter.
4. Use debug mode if you want to inspect the plan and generated GraphQL.
